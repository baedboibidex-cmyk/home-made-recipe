# Home Made Recipe 🍳

A premium cooking app with age verification, user accounts, and international recipes.

## ✨ Features
- verify18+ age gate
- User login/signup
- Free & Premium recipe tiers
- Save favorite recipes
- Responsive design (works on mobile!)

## ▶️ How to Run
1. Clone this repo
2. Open `index.html` in any browser
3. Enjoy cooking!

## 🔒 Premium Content
Unlock foreign recipes (Thai, Japanese, French) and advanced techniques with subscription.
